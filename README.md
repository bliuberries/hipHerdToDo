# hipHerdToDo
This app allows users to create a basic todo-list to keep tabs on and refer back to.

## Technologies
- BootStrap
>Used to create a modal for editing todos

## Database
-MySQL

## Installation

### Installing Dependencies
From within the root directory:

```sh
npm install
```
### Setting up Webpack

```sh
npm run react-dev
```

### Run server
```sh
npm run server-dev
```
